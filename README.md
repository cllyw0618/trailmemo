# TrailMemo 山野手账 — 徒步成长小游戏

一个以 3D 地球为核心的**山野徒步成长手账** Web 应用。TrailMemo 不是记录你去过哪里，而是记录你如何一步步成为真正的山野旅人。

用户可以在交互式 3D 地球上探索全球徒步路线，点亮已完成的路线，通过上传照片和文字自动生成拼贴风格的手账，并在成长系统的陪伴下，从「山野新手」一步步成长为「地球漫游者」。

| 层级 | 技术 |

| 数据库 | SQLite (better-sqlite3) |
| 密码加密 | bcryptjs |

## 功能特性

### 1. 3D 交互地球
- 基于 Three.js 渲染的**真实卫星纹理地球**（地球大气层 2048px 纹理）
- **凹凸贴图（Bump Map）** 带来地形起伏感
- 半透明**云层**以不同速度缓慢旋转，增强真实感
- 多层光照（主光 + 背光 + 补光）营造立体效果
- 地球支持鼠标拖拽旋转、缩放
- 自动缓慢自转动画

### 2. 路线标记系统
- 全球 **18 条经典徒步路线**，覆盖国内十大热门 + 全球经典
- 标记点分为**蓝色（未解锁）** 和 **绿色（已解锁）**
- 每个标记点上方显示**中文地理位置名称标签**
- 标签带切线方向偏移，避免邻近路线重叠
- 标签与标记点之间有细线连接，视觉关系清晰
- 点击标记点直接跳转到手账编辑页

### 3. 用户系统
- 支持**注册**（用户名、密码、性别、生日）
- 支持**登录**
- 密码使用 bcryptjs 加密存储
- 基于 SQLite 的持久化数据存储

### 4. 拼贴手账（Collage Journal）
- 上传照片后，系统自动按**固定模板**排版
- 提供 **5 种拼贴模板**：
  - **单图大版** — 一张大图 + 文字
  - **双图并排** — 两图并列 + 文字
  - **一大一小** — 大图左侧 + 小图和文字右侧
  - **三图网格** — 三图排列 + 文字
  - **满版图文** — 全宽图片 + 文字叠加
- 图片自动压缩至 800px 宽度，优化存储
- 实时预览拼贴效果
- 手账卡片包含：照片、标题、文字、心情标签、天气标签、日期

### 5. 手账展示
- 瀑布流网格展示所有手账
- 卡片风格：拍立得白边、轻微旋转、和纸胶带装饰、小爱心/星星点缀
- 统计面板：已点亮地点数、累计公里数、手账记录数

### 6. 路线索引面板
- 右侧滑出式半透明面板
- 国内 / 全球分组展示
- 芯片式按钮，悬停可预览路线
- 已解锁路线有绿色圆点标记

### 7. 山野成长系统（新增）
- **路线章节系统**：18 条路线分为四个章节
  - 向山而行（4条）：初入山野的入门路线
  - 进入荒野（4条）：更高强度的荒野挑战
  - 朝圣之路（3条）：带有人文信仰意义的路线
  - 世界远方（7条）：全球经典徒步路线
- 每条路线新增故事字段：章节、故事标签、解锁提示、成就分类、情绪关键词、诗意短句
- **成长等级系统**：根据已点亮路线数量自动晋升称号
  - 0 → 山野新手
  - 1 → 初入山野
  - 3 → 草甸行者
  - 5 → 峡谷探索者
  - 8 → 雪山旅人
  - 12 → 远方收藏家
  - 16+ → 地球漫游者
- **旅程进度面板**：在地图页显示当前等级、已点亮数量、距离下一等级的进度

### 8. 轻量成就系统（新增）
- 共 11 个轻量成就，基于已点亮路线自动计算
- 成就包括：第一束山风、草甸收藏家、雪山旅人、峡谷穿越者、海岸漫步者、朝圣者等
- 已解锁成就高亮显示，未解锁成就灰色显示

### 9. 回忆卡片（新增）
- 手账卡片底部增加路线故事信息：旅程章节、情绪关键词、路线短句
- 小字体手写风格，不影响照片展示

### 10. 成长卡与统计（新增）
- 在手账展示页顶部添加「山野成长卡」
- 显示当前称号、已点亮路线、累计公里数、手账记录数量
- 按类型统计已收集路线数量
- 按章节显示进度

## 安装与运行

### 环境要求
- Node.js >= 18
- npm >= 9

### 1. 安装前端依赖

```bash
cd trailmemo
npm install
```

### 2. 安装后端依赖

```bash
cd backend
npm install
```

### 3. 启动服务

需要同时启动前端和后端两个服务：

**启动前端（开发服务器）：**
```bash
cd trailmemo
npm run dev
```
前端运行在 `http://localhost:5173`

**启动后端 API：**
```bash
cd trailmemo/backend
node server.js
```
后端运行在 `http://localhost:3001`

### 4. 打开浏览器

访问 `http://localhost:5173` 即可使用。

## 使用流程

1. **启动页** — 自动过渡到注册/登录页
2. **注册/登录** — 创建账号或登录已有账号
3. **3D 地图** — 浏览全球徒步路线，开启成长之旅
   - 地球自动旋转，可拖拽和缩放
   - 蓝色标记 = 未解锁，绿色标记 = 已解锁
   - 左上角显示「山野旅程」成长面板：当前等级、已点亮路线数、下一等级目标
   - 点击路线弹出预览弹窗，查看路线故事和难度信息
   - 点击「开始记录这段旅程」进入手账编辑
4. **写手账** — 选择模板、上传照片、写下心情
   - 5 种拼贴模板可选
   - 实时预览拼贴效果
   - 可添加心情和天气标签
   - 保存后弹出「路线已点亮」反馈弹窗，显示等级进度
5. **手账展示** — 查看成长和所有手账
   - 顶部「山野成长卡」：当前称号、已点亮路线、累计公里数、手账记录、已收集类型
   - 章节进度条：向山而行 / 进入荒野 / 朝圣之路 / 世界远方
   - 成就系统：11 个成就自动解锁，展开查看
   - 瀑布流展示手账卡片，底部显示路线故事信息（章节、情绪关键词、诗意短句）

## API 文档

### 基础地址
```
http://localhost:3001/api
```

### 注册
```
POST /api/register
Content-Type: application/json

{
  "username": "string (2-20位)",
  "password": "string (至少6位)",
  "gender": "string (男/女/其他)",
  "birthday": "string (YYYY-MM-DD)"
}
```

**响应：**
```json
{
  "id": 1,
  "username": "example",
  "gender": "男",
  "birthday": "2000-01-01"
}
```

### 登录
```
POST /api/login
Content-Type: application/json

{
  "username": "string",
  "password": "string"
}
```

**响应：**
```json
{
  "id": 1,
  "username": "example",
  "gender": "男",
  "birthday": "2000-01-01"
}
```

### 用户列表（调试用）
```
GET /api/users
```

## 设计风格

- **配色**：清新薄荷绿 + 柔和蓝紫色，低饱和度
- **UI 风格**：毛玻璃（Glassmorphism）卡片、圆角、柔和阴影
- **3D 地球**：真实卫星纹理 + 凹凸贴图 + 云层
- **手账卡片**：拍立得边框、轻微旋转、和纸胶带、手绘装饰
- **响应式**：适配桌面端和移动端

## 技术亮点

- **Three.js 3D 渲染**：真实地球纹理、凹凸贴图、多层光照、环境反射
- **智能标签布局**：切线方向偏移算法，避免邻近标签重叠
- **图片自动压缩**：上传时 Canvas 压缩至 800px，优化存储
- **拼贴模板引擎**：CSS Grid 实现 5 种固定布局模板
- **毛玻璃效果**：backdrop-filter 实现半透明磨砂 UI
- **SQLite 持久化**：文件型数据库，零配置
- **成长等级系统**：基于已解锁路线数量自动计算等级和进度
- **轻量成就系统**：前端纯计算成就状态，无需后端支持
- **路线章节系统**：18 条路线按章节分组，展示章节进度
